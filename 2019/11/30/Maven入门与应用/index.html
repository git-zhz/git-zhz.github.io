Maven入门与应用

发表于 2019-11-30 分类于 工具 阅读次数： Valine：

Maven是项目自动化构建工具，它有以下优点：

管理jar包

增加第三方jar包

管理jar包之间的依赖，它会自动将你新增的jar包所依赖的其它jar包导入

将项目拆分成若干个模块

准备

下载安装

1.进入Maven官网：https://maven.apache.org/download.cgi

2.选择apache-maven-x.x.x-bin.zip （x.x.x为相应版本号）

3.解压缩–>E:\apache-maven-x.x.x

配置环境

新建MAVEN_HOME：E:\apache-maven-x.x.x

编辑PATH：添加 %MAVEN_HOME%\bin

验证

1.Ctrl+r 输入cmd进入控制台

2.输入mvn -v 会显示Maven对应版本信息

配置本地仓库

开发工具的设置

1.点击设置Window–>Preferences–>Maven ：

Installations–>add : 选择Maven根目录E:\apache-maven-x.x.x

User Settings–>User Settings Browse : 选择Maven配置文件E:\apache-maven-3.6.3\conf\settings.xml，程序会自动识别并引用本地仓库

使用

构建项目

File–>new–>project–>maven

选择webapp构建项目

目录结构

根目录：工程名

src：源码

main：存放主程序

java：java源码文件

resource：存放框架的配置文件

test：存放测试程序

pom.xml：maven的核心配置文件
