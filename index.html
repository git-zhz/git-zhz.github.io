JSON（JavaScrip Object Notation）：一个轻量级数据交换格式，能够替代XML的工作。

JSON的数据格式比较简单，易于读写，格式都是压缩的，占用带宽小，易于解析。

JSON官网：http://json.org/

下载支持Java的JSON库：https://github.com/stleary/JSON-java

将下载的压缩包解压，打成jar包，引入，这样就可以开始使用JSON了。

Java反射基础

发表于 2019-11-23 更新于 2019-11-24 分类于 基础 阅读次数： Valine：

反射（Reflection）机制是在运行状态中的Java程序对自身进行检查，并直接操作程序的内部属性和方法。

对于任意一个类，都能知道这个类的所有属性和方法。

对于任意一个对象，都能调用它的任意一个属性和方法。

“很多程序架构，尤其是三方框架，无法保证自己的封装是完美的。如果没有反射，对于外部类的私有成员，我们将一筹莫展，所以我们有了反射这一后门，为程序设计提供了更大的灵活性。”

Java加密基础

发表于 2019-10-29 更新于 2019-11-24 分类于 基础 阅读次数： Valine：

异或

可逆

1.同为0，异为1

2.一个数，两次异或之后，是原数本身

“xxxxxx”异或一次：加密

“xxxxxx”再异或一次：解密

MD5

不可逆，速度较快

字符串（字节数组）–>十六进制字符串

引入commons-codec-1.9.jar

SHA256

不可逆，安全性较高

字符串（字节数组）–>十六进制字符串

引入commons-codec-1.9.jar
